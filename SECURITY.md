# Security Policy

## Scope

This is a self-hosted personal tool. It has **no authentication layer** and is not designed for public or multi-user deployment. The intended deployment model is behind a private network (e.g. Tailscale), accessible only to the owner.

If you are running this on a publicly-accessible server without an auth proxy in front of it, that is outside the intended use case and you do so at your own risk.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security concerns.

Instead, email **steven@drkpxl.com** with:
- A description of the issue and potential impact
- Steps to reproduce (if applicable)

I'll respond as quickly as I can for a side project, and will credit you in any fix unless you prefer to stay anonymous.
